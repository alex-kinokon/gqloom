# GQLoom

GQLoom is a GraphQL weaver for TypeScript/JavaScript that weaves GraphQL Schema using Valibot, Zod, or Yup, and supports sophisticated type inference to provide the best development experience.

# @gqloom/federation

This package provides GQLoom support for [Apollo Federation](https://www.apollographql.com/docs/federation).

Read more at [GQLoom Document](https://gqloom.dev/guide/advanced/federation.html).
