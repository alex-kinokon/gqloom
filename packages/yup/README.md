# GQLoom

GQLoom is a GraphQL weaver for TypeScript/JavaScript that weaves GraphQL Schema using Valibot, Zod, or Yup, and supports sophisticated type inference to provide the best development experience.

# @gqloom/yup

This package provides GQLoom integration with [Yup](https://github.com/jquense/yup) to weave Yup Schema to GraphQL Schema.

Read more at [GQLoom Document](https://gqloom.dev/guide/schema-integration/yup).
